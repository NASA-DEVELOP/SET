Installation¶

Skyglow is still in it’s early stages of development, and does not yet offer a double-click installer. As a result running it requires you to setup the development environment, which can be tricky for those not well versed in programming. We’ve tried to be as descriptive as possible in our installation guide for both Windows, Linux and Mac users. If you have any questions or trouble installing, feel free to ask for help from the support page and describe your issue to us.

  • Required Packages

Note

Please read our coding conventions before developing yourself!

Windows¶

  1. download Python Anaconda 2.7

  2. download Git Bash

  3. open a terminal, navigate to the directory you would like to contain the files for the Toolbox, and run git clone <insert link to SkyglowGithubhere>

  4. run (need to double check that this is right)

    conda create env geospatial.yml

  1. run

    python Itest.py

  • These install directions will change once tool is run as an ArcGIs toolbox
